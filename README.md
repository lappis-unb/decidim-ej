# Decidim::Ej

A Decidim component that enables EJ integration..

## Usage

Ej will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-ej"
```

And then execute:

```bash
bundle
```
